## Assignment Details

- **Process:** The assessment involves building responsive layouts, implementing complex animations, and ensuring a premium user experience.

## Getting Started

To run the project locally, install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
