# Olive - Food Scanner App (Interview Assignment)

Welcome to the Olive Food Scanner App repository. This project is a front-end interview assignment focused on creating a high-fidelity, interactive web application using Next.js, React, Tailwind CSS, and Framer Motion. It showcases complex animations, responsive design, and modern UI/UX principles.

## Features

- **Interactive UI**: Dynamic product carousels, fluid animations, and smooth transitions using Framer Motion.
- **Modern Tech Stack**: Built with Next.js, React, and styled with Tailwind CSS.
- **Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **High-Fidelity Components**: Includes detailed components like Hero, How It Works, Health Benefits, and custom curved interfaces.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
